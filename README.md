# PSP-2048

Game "2048" on PSP

一款运行在PSP控制台上的2048。2015年暑假尝试PSP开发时写的。

![ ](./screenshot.png)

## 玩法

* 将`EBOOT.PBP`复制到PSP实机存储卡中，或者使用PPSSPP等模拟器打开。
* 使用上下左右键移动，“□”键撤销，最多撤销16步。
* 不设最高分，但是在所有框框都被填满且无法移动的情况下，游戏结束。
